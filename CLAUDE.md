# CLAUDE.md

## Project

TanStack Start (SSR) + React 19 + Tailwind v4. File-based routing. Bun runtime.

## Commands

```bash
bun run dev       # dev server on :3000
bun run build     # production build
bun run check     # biome lint + format (run before commit)
bun run test      # vitest
```

## Structure

```
src/
  routes/
    __root.tsx        # root layout: sidebar, ThemeProvider, dark-mode script
    index.tsx         # home / tool directory
    json-diff.tsx     # JSON diff tool
    random.tsx        # random number generator
    routeTree.gen.ts  # auto-generated by TanStack Router — do not edit
  context/
    theme.tsx         # ThemeProvider + useTheme hook
  styles.css          # Tailwind import + dark variant config
```

## Key conventions

- **Dark mode**: class-based (`@variant dark` in styles.css). Toggle adds `.dark` to `<html>`. Flash prevention via inline `<script>` in `<head>` reading localStorage.
- **Routes**: Add file → TanStack Router picks it up on next dev start. Also add nav entry to `navItems` in `__root.tsx`.
- **`routeTree.gen.ts`**: Auto-regenerated. If adding routes without restarting dev, update manually (see existing pattern).
- **Imports**: Use `react-syntax-highlighter/dist/esm/prism` (not the barrel). Style imports from `react-syntax-highlighter/dist/esm/styles/prism`.
- **Linter**: Biome. Run `bun run check` before committing.

## Adding a new tool

1. `src/routes/my-tool.tsx` — `createFileRoute('/my-tool')({ component: MyTool })`
2. Add to `navItems` in `__root.tsx` with a lucide icon
3. Dev server auto-regenerates route tree
